## Image

### Using Markdown

[markdown 문법](https://www.markdownguide.org/basic-syntax/#images)을 사용하면 다음 코드를 이용해 이미지를 추가할 수 있습니다.

```md
![title](/path/image.jpg)
```

이미지 파일 크기는 5MB 미만이어야 합니다. 그렇지 않은 경우, [Cloudinary](https://cloudinary.com/) 또는 [S3](https://aws.amazon.com/s3/)와 같은 서비스에 호스팅하는 것을 권장합니다. 그런 다음 해당 URL을 사용하여 임베드할 수 있습니다.

### Using Embeds

To get more customizability with images, you can also use [embeds](/writing-content/embed) to add images

```html
<img height="200" src="/path/image.jpg" />
```

### iFrames

문서 내에 다른 HTML 페이지를 로드합니다. 비디오 임베딩에 가장 일반적으로 사용됩니다.

```html
<iframe src="https://www.youtube.com/embed/4KzFe50RQkQ"> </iframe>
```
