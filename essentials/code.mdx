## Basic

### Inline Code

`단어`나 `구문`을 코드로 표시하려면 백틱(`)으로 묶습니다.

```
`단어`나 `구문`을 코드로 표시하려면 백틱(`)으로 묶습니다.
```

### Code Block

[fenced code blocks](https://www.markdownguide.org/extended-syntax/#fenced-code-blocks)을 사용하여 코드를 세 개의 백틱으로 묶고, 스니펫의 프로그래밍 언어를 선행 백틱 뒤에 붙여 구문 강조를 얻을 수 있습니다. 선택적으로 프로그래밍 언어 뒤에 코드 이름을 작성할 수도 있습니다.

```java HelloWorld.java
class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```
